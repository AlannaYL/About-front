# About-front
 
## 網站說明
  About 是一款展覽資訊整合平台，提供國內外藝文活動及相關資訊，使用者可在網站註冊，並在登入後進行收藏或訂票等功能。

### 使用者前台
  *會員註冊／登入
  *加入購物車／送出訂單功能
  *加入／移出收藏功能
  *查看會員資料
  *查看訂單
  * 查看收藏

### 前台使用技術
  *使用 Quasar 開發、切版
  *Vue Router 頁面切換
  *Sass 預處理器

### 管理者後台
  *上架展覽資訊
  *編輯修改展覽頁面
  *編輯會員資料
  *查看訂單資訊

### 後台使用技術
  *Node.js
  *Express
  *MongoDB
  *gitgub 部署
